# AGENTS.md

## Behavior
- In general, treat the user’s established workflow choices as intentional; only override them when you can articulate a clear, stated benefit.
- For more complex tasks, keep a todo list to ensure you don't miss anything. If you discover something new to investigate while working, add it to the list.
- Consider whether something is strictly required before adding it. Text and code consume human attention which is a valuable resource.
- Keep this file up to date when learning something very important; but keep the file short.
- Strike a balance between token usage/context size and getting all necessary context.

## Code Style Guidelines
- Commit changes to checkpoint your work and deliver logical change bundles. NEVER push.
